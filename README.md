# Rust Learning Projects

A collection of exercises and small projects to learn Rust.

## Projects

-   **[a_hello](./a_hello)**: Basic "Hello, World!" application.
-   **[b_variables](./b_variables)**: Experiments with variable manipulation and randomness.
-   **[c_types](./c_types)**: Exploration of Rust data types.
-   **[d_simple_types](./d_simple_types)**: Examples of simple type usage.

## Usage

To run any of the projects, navigate to the directory and use `cargo run`.

Example:

```bash
cd a_hello
cargo run
```
